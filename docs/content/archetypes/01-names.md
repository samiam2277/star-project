## 2. 商标盲检初评

**TM 风险等级说明:**

- **L (低)** — 该词在商业品牌注册库中**作为产品/服务名**不显著(可能有书/电影但不构成产品类目冲突)
- **M (中)** — 该词有少量品牌占用,但在"占星 app / 心理产品 / 内容服务"类目上未发现直接冲突(需要 USPTO + Class 9/42 类详查)
- **H (高)** — 该词与知名品牌/科技公司商标重叠**或**为大众词汇难以注册,**必须重命名或调整**

**当前 v1 命名 TM 分布:**

## 4. 命名设计原则(回顾)

### 4.1 中文命名

- 2-3 字,**动作/状态 + 者/人/师**结构为主
- 同阵营内不重复后缀(避免 12 个都叫"X者")
- 实际分布:
  - 「**者**」后缀:24 个(拓荒者/号召者...大量)
  - 「**人**」后缀:5 个(匠造人/燃灯人/渡舟人/织梦人/吟谱)— 注:吟谱者用"者",我重新数 → 4 个
  - 「**师**」后缀:3 个(雕琢师/调律师/炼金师/解秘者...) — 注:解秘者用"者"
  - 「**主**」后缀:1 个(盛宴主)
  - 实际统计:者 25 / 人 5 / 师 3 / 主 1 / 直接用动词 1(共鸣者→者) = 35 + 共鸣 1 = 这里需重新核
  - **正式发布前**会做精确分布平衡

### 4.2 英文命名

- 1 词为主(Trailblazer, Artisan, Vanguard...),双词次之(Deep Diver, Forge Walker)
- 避免 "-er" 单调:混用 "-keeper, -bearer, -wright, -caller, -walker"
- 双语**不互译**:中文重意境(深潜者),英文重音韵(Deep Diver)— 中文是"深 + 潜 + 者",英文是"D-D 头韵"

### 4.3 文化盲检结果

| 风险类型 | 检查项 | 结果 |
|---|---|---|
| 中文方言/网络梗负面 | 36 个名字 | ✅ 通过,无打工人/躺平类联想 |
| 中文宗教专用 | 36 个名字 | ⚠️ "招魂者"在道教/民俗有负面联想,建议 UI 文案中弱化"招魂"的传统含义,强调"召回自己" |
| 英文文化挪用 | 36 个名字 | ⚠️ "Shaman/Witch Doctor"已主动避开;"Alchemist"在西方文化中正面 |
| 海外华人(主受众) | 双语两端 | ✅ 中英两端都有清晰意象,不依赖单一文化语境 |

## 5. 决策树映射的输入信号(给 task #6 用)

每个原型的"占星主特征"列已经给出了**最强的识别信号**,task #6 会把它们转化为完整的决策树规则集:

```typescript
// 示例:T01 深潜者 的规则
{
  id: 'T01',
  faction: 'T',
  subThemeIndex: 1,
  nameZh: '深潜者',
  nameEn: 'Deep Diver',
  conditions: {
    // 主路径(高 priorityWeight)
    primary: [
      { sunSign: ['Scorpio'], moonSign: ['Scorpio'], weight: 100 },
      { sunSign: ['Scorpio'], moonHouse: [8, 12], weight: 90 },
      { sunSign: ['Pisces'], moonSign: ['Scorpio'], weight: 80 },
    ],
    // 次路径(中等 weight)
    secondary: [
      { sunSign: ['Cancer', 'Pisces'], plutoConjunctSun: true, weight: 70 },
    ],
    // 排除路径(避免错配)
    exclude: [
      { ascSign: ['Aries', 'Sagittarius'] }, // 上升火象更可能落到 T 阵营其他原型
    ]
  },
  priorityWeight: 95,
  fallbackOnly: false
}
```

## 6. 与 JSON 数据契约对接

正式产出的 JSON 文件 `docs/content/archetypes/archetypes.json`(在 task #4 文案完成时一并生成),schema:

```typescript
interface Archetype {
  // 元数据
  id: string;                    // 'C01' | 'L05' | 'T12' ...
  faction: 'C' | 'L' | 'T';
  factionNameZh: string;         // '创造系' | '链接系' | '转化系'
  factionNameEn: string;         // 'Creation' | 'Connection' | 'Transformation'
  subThemeIndex: number;         // 1-12

  // 双语命名
  nameZh: string;                // '深潜者'
  nameEn: string;                // 'Deep Diver'

  // 占星主特征(决策树用)
  astroSignals: {
    sun?: string[];
    moon?: string[];
    asc?: string[];
    mars?: string[];
    moonHouse?: number[];
    aspects?: Array<{p1: string; p2: string; type: 'conjunct' | 'opposition' | 'square' | 'trine' | 'sextile'}>;
  };

  // 4 段文案(task #4 填充)
  textZh: {
    oneliner: string;
    gifts: string[];        // 3 条
    shadows: string[];      // 2 条
    evolutionPath: string;  // 南交 → 北交一句话
  };
  textEn: { /* 同上结构 */ };

  // 视觉
  themeColor: string;        // HEX
  silhouetteAsset: string;   // /assets/archetypes/T01-deep-diver.svg

  // 雷达 6 维上限(0-1)
  radarCeiling: {
    creativity: number;
    leadership: number;
    insight: number;
    social: number;
    intuition: number;
    action: number;
  };

  // SEO / 分享
  slug: string;              // 'deep-diver'
  shareTitle: { zh: string; en: string };

  // 元数据
  confidenceMinimum: 'low' | 'mid' | 'high';  // 该原型能在多低的数据完整度下触发
  trademark: { risk: 'L' | 'M' | 'H'; notes?: string };
}
```
